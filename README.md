### Rodar localmente

- **Opção 1 (simples)**: abra `index.html` no navegador.
- **Opção 2 (recomendado)**: rode um servidor local (evita problemas de CORS/áudio/vídeo):

```bash
python -m http.server 5173
```

Depois abra `http://localhost:5173`.

### Controles

- **Mover**: `W A S D`
- **Atacar**: `J` ou **click**
- **Habilidade 5/6**: `5` / `6`
- **Interagir**: `E`
- **Inventário**: `I`
- **Missões**: `Q`
- **Dash**: `Space`
- **Pausar**: `Esc`

### Extras

- No menu inicial: **EXTRA → Panteão**
  - **Nível 1**: sequência de bosses (todas as habilidades liberadas)
  - **Nível 2**: *no-hit* + **30 minutos**

### Códigos

- **OPÇÕES → código**: `debugmode`
  - Dentro do jogo: `F1` abre o menu dev (warp/itens/level/invencibilidade).
